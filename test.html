Be Hoppy is a single page application where users build lineups of sports rosters. After sign in, user's have access to all of their past lineups. User's can choose to create new lineups by selecting a player from the preselected list of all time greats. The back-end was built using a combination of Express, Mongoose, and PostgreSQL. The front-end was built using React.js, utilizing Axios for http requests to communicate with the API.
